## 关于项目
* 这个项目的目的是帮助相关人员更快的了解数字和模拟继集成电路和AI加速器硬件实现相关 的知识。项目整理的论文包括各个顶会(ISCA, MICRO, ASPLOS, HPCA)以及IEEE收录的相关文章，并对其的主要内容和创新点进行了概括，方便快速查找和了解。

* 这个项目由GLW算法组提出并进行相关的维护和更新。如果您有任何好的建议和问题，可以在GitHub上与我们一起讨论。
